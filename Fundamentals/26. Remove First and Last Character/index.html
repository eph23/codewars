JavaScript Exercise – Remove First and Last Character

Your goal is to write a function that removes the first and last characters of a string. You're given one parameter, the original string. Important: Your function should handle strings of any length ≥ 2 characters. For strings with exactly 2 characters, return an empty string.

Important: Your function should handle strings of any length ≥ 2 characters. For strings with exactly 2 characters, return an empty string. Examples 'eloquent' --> 'loquen' 'country' --> 'ountr' 'person' --> 'erso' 'ab' --> '' (empty string) 'xyz' --> 'y'

Test Data

  • removeChar('eloquent') // 'loquen'
  • removeChar('country') // 'ountr'
  • removeChar('person') // 'erso'
  • removeChar('place') // 'lac'
  • removeChar('ooopsss') // 'oopss'
